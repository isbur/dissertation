Главная книга.xlsm - вроде действительно главная, можно этому верить. Я надеюсь.

Как пользоваться:
	1. Запустить скрипт CreateWord.bat
	2. Открыть Главняа книга.xlsm
	3. Отредактировать Лист2 довольно очевидным образом
	4. В скрипте Pies поменять значение переменной inTables на соответствующее числу таблиц, перечисленных на Листе2
	5. Открыть все интересующие таблицы
	6. Запустить скрипт Pies
